# Beecrowd 2059 — Par ou Ímpar
## Editorial Técnico Completo

### Enunciado (resumo)
Dois jogadores escolhem **PAR** ou **ÍMPAR**. Dados dois números, determine o vencedor.

---

## Ideia Central
A soma define o vencedor:
- Par → jogador que escolheu PAR
- Ímpar → jogador que escolheu ÍMPAR
